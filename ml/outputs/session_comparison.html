🛠 어떻게 만들어졌나요?

이 파일은 ml/compare_sessions.py로 자동 생성되었습니다.

데이터 흐름:
1. chewing-vision CLI — MediaPipe FaceLandmarker로 영상에서 MAR(입 열림 비율) 추출 → 1초 window GT 라벨 생성
2. IMU 전처리 (ml/utils.py · make_windows_with_times) — AirPods 50Hz 신호를 2초/0.5s stride 슬라이딩 윈도우로 분할, 6축×(RMS+Std) = 12차원 피처 추출
3. Random Forest (sklearn, n=20) — time-based 80/20 split으로 학습·평가
4. 씹기 카운팅 — ML 예측 결과에서 연속 chewing 윈도우를 bout 단위로 묶어 duration × 1.2 Hz로 씹기 수 추정
5. 시각화 (matplotlib) — F1 바차트, IMU 신호+GT라벨, 예측 타임라인, Confusion Matrix
6. 이 HTML — Python f-string으로 생성 (ml/compare_sessions.py)

재실행: .venv/bin/python ml/compare_sessions.py
